O projeto tem como finalidade realizar uma análise de orçamento de capital a partir de dados reais obtidos no site do Clube Atlético Mineiro, incluindo estimativas de dados dos quais não foi possivel obter. 

A análise foi feita em duas etapas:

* Análise de Cenários: foi feita a partir da obtenção de dados a serem utilizados no cenário base da simulação 

* Análise de Sensibilidade: Permite avaliar a variação ocorrida no valor do projeto (VPL), dada a alteração proporcional no valor de uma variável mantendo tudo constante. Quanto maior a inclinação da curva que representa cada variável, mais sensível é o VPL a uma alteração em uma unidade dessa variável. 

* Simulação de Monte Carlo: É uma ferramenta relevante pois possibilita a atribuição de distribuições de probabilidade a variáveis que serão projetadas para o futuro, considerando assim uma medida de risco no recebimento dos fluxos de caixa esperados. A partir dela pode-se calcular as estatísticas descritivas esperadas para aquele projeto, incluindo medidas de risco no recebimento dos fluxos de caixa.
